# Contributing to IFSC Finder India

First off, thanks for taking the time to read this and considering contributing!

## How Can I Contribute?

### Reporting Bugs
If you find a bug in the source code or an issue with the bank data, please check the existing issues to avoid duplicating efforts. If the issue is new, please open a new issue and include:
- A clear description of the problem.
- Steps to reproduce the bug.
- Environment details (Browser, OS, Mobile, etc.).

### Suggesting Enhancements
Have an idea for a new feature or an improvement? Open an issue and let us know! Please provide clear details about the feature and why it would be beneficial to the platform.

### Contributing Code
1. **Fork the repository** on GitHub.
2. **Clone the project** to your local machine.
3. **Create a new branch** for your feature or bugfix (`git checkout -b feature/your-feature-name` or `bugfix/issue-description`).
4. **Make your changes** ensuring you follow our Mobile-First and strict performance guidelines.
5. **Commit your changes** with clear and descriptive commit messages.
6. **Push to your fork** and submit a **Pull Request (PR)** against the `main` branch.

## Code Guidelines
- Keep accessibility in mind (score 90+ target).
- Prioritize load times and performance (< 2s load time).
- Ensure a responsive, clean, Mobile-First layout.
- Maintain consistency with the pre-existing UI patterns.
